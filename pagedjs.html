Code Examples

Technical documents often require code examples. This chapter demonstrates how code blocks are formatted and displayed.

HTML Example

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>PagedJS Example</title> <style> @page { size: A4; margin: 2cm; } </style> </head> <body> <h1>Hello, PagedJS!</h1> </body> </html>

CSS Example

/* Page setup for print */ @page { size: A4 portrait; margin: 2.5cm 2cm; @top-center { content: "Document Title"; font-size: 12px; } @bottom-right { content: "Page " counter(page); font-size: 10px; } } /* Chapter styling */ .chapter { page-break-before: always; counter-increment: chapter; } .chapter h1::before { content: "Chapter " counter(chapter) ": "; }

JavaScript Example

// PagedJS CLI usage example const { exec } = require('child_process'); function generatePDF(inputFile, outputFile) { const command = `pagedjs-cli ${inputFile} -o ${outputFile}`; exec(command, (error, stdout, stderr) => { if (error) { console.error(`Error: ${error}`); return; } console.log(`PDF generated: ${outputFile}`); }); } // Usage generatePDF('test.html', 'output.pdf');

Recommendations

When using PagedJS CLI for document generation:

  1. Test your CSS thoroughly with the specific content you'll be using
  2. Pay attention to page break behavior with different content types
  3. Use the page-break-inside: avoid property for content that should stay together
  4. Consider using custom page types for different sections of your document
